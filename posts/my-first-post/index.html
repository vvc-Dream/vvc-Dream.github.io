用Hexo+Github搭建你的个人博客

总述

搭建的过程分为以下几步：

  1. 安装Node.js
  2. 安装Git软件
  3. 注册Github账号
  4. 创建项目
  5. 配置本地密匙
  6. 安装hexo博客框架
  7. 测试运行
  8. 上传到Github

安装Node.js

Node.js下载地址：Node.js

安装一路默认安装即可

详细安装文档参看：Node.js安装文档

安装Git软件

Git软件下载地址：Git

安装一路默认安装即可

关于更多Git详情参看：Git教程

安装Hexo

1. 什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页.

2. 安装Hexo

新建一个文件夹Hexo，在该文件夹下右键，点击 Git Bash Here ，打开Git命令窗口.

安装Hexo命令:

npm install -g hexo-cli  #-g表示全局安装

提示：在Git命令窗口无法使用Ctrl+V粘贴，可右键，点击 Paste .
